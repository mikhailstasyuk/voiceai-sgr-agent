# Glossary

- **ASR**: Automatic Speech Recognition; converts incoming audio to text.
- **VAD**: Voice Activity Detection; marks speech/silence transitions.
- **Barge-in**: Interrupting assistant output when user starts speaking.
- **Turn**: One user utterance and the assistant response sequence.
- **Segment**: A partial chunk of synthesized assistant audio.
- **Realtime session**: Active WebSocket conversation between client and backend.
- **Boundary validation**: Parsing and validating external/input/config data before internal use.
- **Structured output**: LLM response constrained to expected schema/shape.
- **Agent-first workflow**: Humans define intent; Codex executes implementation and verification.
- **Reference repository**: `sgr-agent-core/`; examples only, not runtime source of truth.
