# Plans

Use plans for substantial multi-step work.

## Directories
- `active/`: plans currently in execution
- `completed/`: archived plans with outcomes

## Plan Template
Each substantial plan should include:
- Goal
- Context
- Assumptions
- Constraints
- Implementation Steps
- Validation Plan
- Progress Log
- Decisions / Tradeoffs
- Follow-Ups / Debt

Use [active/plan-template.md](/home/rhuu/mygit/hypercheap-voiceAI/docs/plans/active/plan-template.md) as a starter.

## Lifecycle
1. Create plan in `active/` before non-trivial implementation.
2. Update progress log and decisions during work.
3. Move to `completed/` when done; add final validation summary and remaining debt.
