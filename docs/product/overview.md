# Product Overview

Hypercheap VoiceAI is a realtime voice assistant stack focused on low latency and low operating cost.

## Primary Experience
- User clicks mic in the web UI.
- User speaks naturally.
- Assistant responds with streamed voice output.
- User can interrupt assistant speech by speaking again.

## Primary Value
- Low-latency turn handling
- Lower-cost provider composition
- Browser-based client with simple local setup
